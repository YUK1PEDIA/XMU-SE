# 马原

- 思政课里含金量最高的一门，因为马原有成熟的体系，相较于其他思政课更好背，而且不是死记硬背，理解之后留个印象就能分析做题

- 22 级论述题考察内容：

  1. 根据**联系和发展的基本环节中的现实与可能**结合材料分析
  2. 唯物史观
  3. 价值论与唯物史观
  4. 剩余价值理论
